## ðŸš¦ Usage
1. Enter API endpoint URL:
```
https://api.example.com/data
```

2. Click **Fetch Data** button

3. View formatted JSON response:
```json
{
  "data": {
    "image": "https://example.com/image.jpg"
  }
}
```

4. For image preview:
```
Enter JSON path: data.image
```
